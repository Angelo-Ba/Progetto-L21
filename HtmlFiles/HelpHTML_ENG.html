Function Study Project - Help

Supported Function

These are the supported functions. Note that arg can be simply (x) or something more complex:
  • sin(arg)
  • cos(arg)
  • tan(arg)
  • ln(arg)
  • logBase(arg)

CaseInsestive

This application is caseInsestive, i.e. inserting SIN (x) is equivalent to sin (X).

Button Review

A short list of the functions performed by each button in the menu.
  • Aggiungi->Nuova Funzione : to plot a function
  • Aggiungi->Funzione Salvata->aggiungi : to memorize some functions
  • Visualizza->zeri : to show the zeros on the graph
  • Visualizza->estremanti : to show the extremes on the graph
  • Visualizza->coordinata puntata : to show the coordinate pointed to by the cursor
  • Visualizza->opzioni cromatiche : to set your favorite cormatic settings
  • Esporta->istantanea : to export a snapshot of the graph

Function Syntax

The syntax for inserting functions foresees to use "x" as an independent variable and always use round brackets to indicate a function (i.e. tan (x + 2)).

Saved Function

You can use saved functions as arguments to build more complex functions to draw. To do this, just specify the name chosen for the saved function, followed by the round brackets with an x.
For example:
saved function: a = x + 2
function to plot: sin (a (x)) == sin (x + 2)

Keyboard shortcuts

You can also use keyboard shortcuts to improve the use of the program:
  • CTRL && +/- zoom in / out/
  • CTRL && [arrow key]: pan left-right / up-down
  • CTRL && H: return home (ie: default perspective on the graph)
  • CTRL && P: plot a new function on the graph.
  • CTRL && E: export a snapshot of the current graph
  • CTRL && X: view critical points. (eXtremanti)
  • CTRL && Z: view Zeros
  • CTRL && C: clears the graph.

Shell Commands

If you are using the application from a shell, you should know the following commands:
  • help-sh: displays these shell help instructions.
  • plot [expression]: plots a new function on the graph.
  • del [expression]: removes all of the functions with a given expression.
  • deriv [expression]: derives the expression and plots the derivative.
  • help: displays the help dialog-box.
  • hdls: de/activates the 'headless' mode (headless mode disables graph-frame popups).
  • show: opens up the graph-frame if it was hidden.
  • smpl [expression]: attempts to simplify an expression.
  • eval [expression]: evaluates a constant expression.
  • exit: terminates the program
