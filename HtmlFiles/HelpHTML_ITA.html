Progetto studio di funzioni - Help

Funzioni supportate

Queste elencate sono le funzioni supportate. Nota che l'argomento può essere (x) o qualcosa di più semplice:
  • sin(arg)
  • cos(arg)
  • tan(arg)
  • ln(arg)
  • logBase(arg)

Case-Insensitive

Questa applicazione è case-insensitive, ovvero inserire SIN (x) equivale a sin (X).

Bottoni sul Menu

Un breve elenco delle funzioni svolte da ciascun pulsante nel menu:
  • Aggiungi->Nuova Funzione : per aggiungere una nuova funzione
  • Aggiungi->Funzione definita da utente->aggiungi : memorizzare delle funzioni
  • Aggiungi->Cronologia : mostrare la cronologia delle funzioni inserite
  • Visualizza->zeri : mostrare gli zeri sul grafico
  • Visualizza->estremanti : mostrare gli estremanti sul grafico
  • Visualizza->coordinata puntata : mostrare sul grafico la coordinata puntata dal mouse
  • Visualizza->opzioni cromatiche : impostare le preferenze grafiche
  • Visualizza->passo : impostare il passo per il calcolo dei Samples
  • Esporta->istantanea : esportare una foto del grafico

Sintassi delle funzioni

La sintassi per l'inserimento delle funzioni prevede di utilizzare "x" come variabile indipendente e di utilizzare sempre parentesi tonde per indicare una funzione (es. tan (x + 2)).

Funzioni definite dall'utente

E' possibile utilizzare le funzioni salvate come argomenti per creare funzioni più complesse da disegnare. Per farlo è sufficiente specificare il nome scelto per la funzione salvata, seguito dalle parentesi tonde con una x.
Per esempio:
funzione definita dall'utente: a = x + 2
funzione da plottare: sin (a (x)) == sin (x + 2)

Scorciatoie da tastiera

Puoi anche utilizzare le scorciatoie da tastiera per migliorare l'utilizzo del programma:
  • CTRL && +/- zoom in / out/
  • CTRL && [arrow key]: movimento sinistro-destro / sopra-sotto
  • CTRL && H: ritorno alla home (es: prospettiva di default sul grafico)
  • CTRL && P: disegnare nuova funzione sul grafico
  • CTRL && E: esportare una istantanea grafico
  • CTRL && X: mostra estremanti (eXtremanti)
  • CTRL && Z: mostra zeri
  • CTRL && C: svuoatare il grafico.

Comandi da shell

Se utilizzi il programma da shell, dovresti conoscere i seguenti comandi:
  • help-sh: mostra le informazioni
  • plot [expression]: disegna una nuova funzione sul grafico
  • del [expression]: rimuove tutte le funzioni con quella espressione
  • deriv [expression]: deriva l'espressione e la plotta
  • help: mostra la finestra di help
  • hdls: attiva/disattiva la 'headless' mode (headless mode disabilita i frame pop-up).
  • show: mostra il graphFrame se era nascosto
  • smpl [expression]: semplificare l'espressione
  • eval [expression]: valuta l'espressione
  • exit: chiude il programma
